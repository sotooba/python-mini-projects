# Python Mini-Projects

This repository contains a collection of beginner-friendly mini projects designed to help you understand core programming concepts through simple, practical examples.

## What is Included?

- **ASCII Art Generator** – A program that takes user input (text and font) and displays the output in the chosen font.
- **Bee Game** – A classic “guess the word” game with a scoring system.
- **Calculator** – Not just any basic (+, −, ×, ÷) calculator — this one also validates and handles user input.
- **Calendar** – A program that takes a date, month, and year from the user and displays the formatted calendar for the selected month and year.
- **Calories** – A program that takes a fruit or vegetable as input and displays the calories per 100g.
- **Guessing Game** – A number-guessing game with a scoring system.
- **Line Count** – Takes a filename as a command-line argument and counts the total number of code lines, empty lines, and comments in that file.
- **Mind Reader** – A simple mind-reading game.
- **Minutes** – Takes a date input from the user, validates it, and displays the number of minutes between that date and the current date.

These projects use various Python libraries and are great for practicing foundational programming skills.
